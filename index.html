What is a Digital Dead Drop?

A digital dead drop is a method of indirect communication where one person leaves information at a location and another person collects it later without direct contact.

Purpose of This Project

This project is an educational simulator created to explain how attackers and defenders understand covert communication.

Disclaimer: This project is an educational simulation only. It does not perform real secure communication.
